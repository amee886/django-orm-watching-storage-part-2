# Пульт охраны банка

Это внутренний репозиторий для сотрудников банка "Сияние". Если вы попали в этот репозиторий случайно, то вы не сможете его запустить, т.к. у вас нет доступа к БД, но можете свободно использовать код вёрстки или посмотреть как реализованы запросы к БД.

Пульт охраны - это сайт, который можно подключить к удалённой базе данных с визитами и карточками пропуска сотрудников нашего банка. Поможет отследить подозриельных пользователей и посмотреть какое время они находились на сайте и кто находится в хранилище в данный момент.


### Как установить 

1. **Клонировать репозиторий**
```bash
git clone https://github.com/amee886/django-orm-watching-storage-part-1.git cd django-orm-watching-storage-part-2
```
2. **Установите зависимости**
 ```bash
   python -m pip install -r requirements.txt
```
3. **Получите данные для базы данных**
  
Получите данные для базы данных(вам понадобится хост для подключений, порт, имя базы, имя пользователя и пароль,секрертный ключ)

Создайте файл .env в корневом каталого проекта и добавьте туда эти данные

Пример содержимого
```bash
DB_HOST=ваш хост для подключения
DB_PORT=ваш порт для подключения
DB_NAME=имя базы
DB_USER=имя пользователя
DB_PASSWORD=пароль
DB_SECRET_KEY=секретный ключ
DEBUG=значение True или FAlse(по дефолту False) 
```

4. **Запустите программу**

```bash
python manage.py runserver 0.0.0.0:8000
```
### Функции

**get_duration(visit)** - функция позволяющая узнать сколько времени провел на сайте человек

**format_duration(duration)** - функция переводящая неудобное обозначение времени в часы и минуты

**is_visit_long(visit, minutes=60)** - функция позволяющая узнать кто находился на сайте больше какого определенного времени(в данной функции больше часа)
